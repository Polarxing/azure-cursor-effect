# 蔚蓝光标特效 · Chrome 浏览器插件

蔚蓝主题的鼠标交互特效插件，包含点击特效和长按拖尾特效，支持自定义参数。

## ✨ 功能特性

### 点击特效
- 淡蓝色圆快速放大后渐隐
- 三段随机位置的蓝白色柔光弧线
- 四个高频闪烁的三角粒子发散飞出
- 弧线半径跟随圆变化，被圆推着向外扩展

### 拖尾特效
- 冰蓝单线拖尾，由粗到细自然消失
- 沿轨迹稀疏分布的白蓝三角粒子
- 三角向轨迹两侧轻微发散
- **拖尾柔光效果**：外层淡光晕 + 内层亮线的双层结构

### 控制面板
- 开关特效
- 实时调节所有参数
- 参数自动保存（同步到 Chrome 账号）
- 一键恢复默认设置

## 📦 安装方法

### 开发者模式安装
1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本插件文件夹（`azure-cursor-effect`）
5. 安装完成，插件图标会出现在工具栏

## 🎮 使用方法

### 触发特效
- **点击鼠标**：按下瞬间触发点击特效（圆+弧线+三角）
- **按住滑动**：按住鼠标左键移动，触发拖尾特效
- **松开鼠标**：拖尾特效停止

### 调节参数
1. 点击工具栏的插件图标，打开控制面板
2. 拖动滑块调节参数，实时预览效果
3. 参数会自动保存，刷新页面后依然生效
4. 点击「恢复默认设置」可重置所有参数

## ⚙️ 可调参数

### 点击特效 · 圆
- 圆最大半径
- 放大时间
- 消失时间

### 点击特效 · 弧线
- 弧线数量
- 每段角度
- 弧线粗细
- 柔光强度

### 点击特效 · 三角
- 三角数量
- 三角大小
- 飞行距离
- 闪烁频率

### 拖尾特效 · 线
- 拖尾长度
- 起点粗细
- 柔光扩散
- 柔光强度

### 拖尾特效 · 三角
- 三角最大间距
- 三角最小间距
- 三角初始大小
- 发散速度

## 🔧 技术说明

- 使用 Canvas 2D 绘制，高性能
- 基于 requestAnimationFrame 的动画循环
- 四边形填充法绘制渐变宽度线段，无缝衔接
- 双层柔光结构（外层淡光晕 + 内层亮线）
- 支持触屏设备
- 使用 chrome.storage.sync 同步配置

## 📝 注意事项

1. 插件默认在所有网站生效
2. 如果特效不显示，请刷新页面
3. 参数调节后实时生效，无需刷新
4. 配置会同步到你的 Chrome 账号
5. 部分特殊网站（如 Chrome 内部页面）可能无法注入特效
